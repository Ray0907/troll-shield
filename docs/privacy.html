Privacy Policy 隱私權政策

我們非常重視您的隱私，並希望對我們如何收集和使用您的資料保持透明。此隱私權政策說明當您使用「AI 酸民護盾」(以下簡稱「擴充功能」)時，我們收集哪些資料以及如何使用這些資料。

擴充功能收集哪些資料

當您使用擴充功能時，我們可能會收集以下資料：

  • 自訂 Prompt：擴充功能會將使用者提供的 Prompt 存儲在客戶端。這些資料用於產生發送給 Open AI 的訊息。
  • API 金鑰：擴充功能會需要使用者輸入 OpenAI 的 API Key，以通過 API 生成回應。這些資訊僅存儲在客戶端。
  • 網站數據：擴充功能會讀取當下網站內容以生成摘要。這些數據僅用於生成回應，並不會被我們存儲。請參閱 OpenAI 的隱私權政策，以了解他們如何使用 API 共享資料。

擴充功能與誰共享您的資料

用戶的 Prompt 與當下的網頁資料會發送給 OpenAI API 請求產生回應。請參閱 OpenAI 的隱私權政策，以了解他們如何使用 API 共享資料。

鏈接到其他網站

設定頁面包含鏈接到其他未由我們運營的網站，大多是作者的社群網站與 Open AI 設定頁面。如果您點擊第三方鏈接，您將被導向該第三方的網站。我們建議您查看每個您訪問網站的隱私權政策。我們對任何第三方網站或服務的內容、隱私政策或做法不承擔任何責任。

您的選擇

您可以隨時選擇禁用或卸載擴充功能。您也可以清除您的客戶端資料。
